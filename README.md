Pokemon finder
=======================


## Install

Clone this repository

```shell
$ git clone https://github.com/eesnaola/pokemon-finder
```

Install dependencies

```shell
$ composer install
```

Run a Web Server


```shell
$ php bin/console server:start
```

This starts the web server at `localhost:8000`

## Unit Testing

Unit test could be executed using:

```shell
$ ./bin/phpunit
```
